# b13-A5

1. 
Q: What is the difference between var, let, and const?

A: var is function-scoped and can be redeclared. let and const are block-scoped (only exist within { }). let allows you to reassign values, while const is for values that should stay the same.

2. 
Q: What is the spread operator (...)?

A: It is a syntax used to "unpack" elements from an array or properties from an object. It is commonly used to create copies of data or merge multiple arrays and objects together.

3. 
Q: What is the difference between map(), filter(), and forEach()?

A: map() creates a new array by changing every item; filter() creates a new array containing only items that pass a specific test; forEach() loops through the items to perform an action (like logging) but returns nothing.

4. 
Q: What is an arrow function?

A: A compact way to write functions using the => syntax. They are shorter than traditional functions and "inherit" the this value from the code around them, which is very helpful in object-oriented programming.

5. 
Q: What are template literals?

A: Strings defined with backticks (`) instead of quotes. They allow you to plug variables directly into text using ${variable} and support text that spans multiple lines easily.
